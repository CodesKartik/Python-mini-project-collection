# Python-mini-project-collection
🧠 Python Mini Projects Collection
📌 Overview

This repository contains three simple Python programs designed for beginners. Each script demonstrates basic concepts like user input, conditional statements, and working with time.

📂 Projects Included
🚦 1. Traffic Light Suggestion

A program that suggests actions based on traffic light color.

💡 Features

Takes user input (red, yellow, green)

Outputs appropriate driving action

▶️ Example
enter the traffic light: red
stop
🧠 Concepts Used

input()

if-elif-else

String handling (lower())

🌦️ 2. Weather Suggestion

Provides suggestions based on current weather conditions.

💡 Features

Accepts weather input (sunny, rainy, cloudy)

Suggests appropriate actions (e.g., sunglasses, umbrella)

▶️ Example
enter the weather: rainy
take an umbrella
🧠 Concepts Used

Conditional statements

User input handling

⏰ 3. Timestamp with Python

Displays the current time in different formats.

💡 Features

Prints:

Full time (HH:MM:SS)

Hour

Minutes

Seconds

▶️ Example Output
14:35:20
14
35
20
🧠 Concepts Used

time module

strftime() function

⚠️ Improvements Suggested

Fix spelling mistakes:

"weater" → "weather"

"i do't know" → "I don't know"

Avoid repeating import time multiple times

Add comments for better readability

Handle invalid inputs more gracefully

🚀 How to Run

Make sure Python is installed

Run any script:

python traffic_light.py
python weather_suggestion.py
python timestamp.py
🎯 Learning Outcomes

By working with these scripts, you will learn:

Basic Python syntax

Conditional logic

Handling user input

Working with time and formatting
